Masterkey

Role: Designer and programmer.
Year: 2022.
Genre: Third-Person, Action, Arcade, Retro, Text-Based.
Platform: PC.

Description
Welcome to MASTERKEY, a multiplayer game where you need to
type in order to play. From accessing settings and menus to
playing the very game itself, your keyboard is your closest
ally.

Speed and precision is key. There is no room for mistakes.

This project was made for my final semester and is now
abandoned. You can find the game here.
